# APCI-IDS-Benchmark
APCI-IDS Benchmark: A Complexity- and Effort-Oriented Benchmark for Computational Software Effort Estimation in Intrusion Detection System Development

This repository contains the APCI-IDS Benchmark developed for the paper:

APCI: A Complexity-Aware Framework for Computational Software Effort Estimation in Intrusion Detection System Development

The benchmark contains 1,040 IDS development project instances generated from multiple public intrusion detection datasets, dataset scales, feature configurations, learning algorithms, and computational resource measurements.
